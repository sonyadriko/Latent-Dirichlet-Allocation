📖 Cara Penggunaan

  1. Masukkan nama project untuk identifikasi analisis Anda.
  2. Siapkan file .txt berisi daftar URL berita (satu URL per baris).
  3. Upload file melalui tombol "Upload Link".
  4. Tentukan jumlah topik yang diinginkan (default: 5).
  5. Klik tombol "Crawling" untuk memulai proses.
  6. Tunggu hingga seluruh proses KDD selesai secara otomatis.
  7. Lihat hasil visualisasi di halaman Visualisasi.
