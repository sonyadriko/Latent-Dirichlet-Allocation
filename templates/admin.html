Document Search & LDA Analysis

Search documents and find similar content using LDA topic modeling

📖 Cara Penggunaan (Mode Otomatis)

  1. Masukkan judul dokumen yang ingin dicari.
  2. Atur jumlah topik dan parameter lain.
  3. Klik "Search Documents" untuk mencari dokumen terkait.
  4. Klik "Train LDA Model" untuk melatih model.
  5. Lihat hasil visualisasi setelah training selesai.

📖 Cara Penggunaan (Mode Upload File)

  1. Masukkan nama project untuk identifikasi analisis.
  2. Siapkan file .txt berisi daftar URL berita (satu URL per baris).
  3. Upload file melalui tombol "Upload Link".
  4. Klik "Upload & Crawling" untuk crawl URL.
  5. Klik "Train LDA Model" untuk melatih model.
  6. Tunggu hingga seluruh proses KDD selesai.
  7. Lihat hasil visualisasi setelah training selesai.
